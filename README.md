# 💫 About Me:
I’m Joyce, a BSc Information Technology student at Eduvos in Cape Town with a strong interest in cybersecurity, network engineering, cloud technologies, and modern web development.<br><br>I enjoy building practical, real-world IT solutions and continuously improving my technical skills through hands-on projects. My experience includes developing enterprise multisite network infrastructures in Cisco Packet Tracer with VLANs, OSPF routing, ACLs, NAT, DHCP, HSRP redundancy, wireless management, and secure network segmentation.<br><br>Beyond networking and security, I also build modern web applications and responsive user interfaces using React, Tailwind CSS, Vite, and JavaScript. I enjoy combining technical problem-solving with clean and user-friendly design.
