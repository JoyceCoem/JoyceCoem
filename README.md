### 💫 About Me

Final-year BSc Information Technology student at Eduvos, Cape Town, specializing in **Security and Network Engineering**. I build practical, real-world IT solutions from enterprise multisite network infrastructures to secure, modern web applications.

- 🔐 Enterprise network design in Cisco Packet Tracer: VLANs, OSPF, ACLs, NAT, DHCP, HSRP redundancy, wireless management, and network segmentation
- 🌐 Modern web development with Angular, React, ASP.NET Core,and Tailwind CSS
- ☁️ Growing focus on cloud security and Azure deployment
- 🎓 deployed a full-stack portfolio (Angular 17 + ASP.NET Core + SQL Server) to Azure
- 🗣️ Former SRC Representative — hosted technical workshops for fellow students
